# Vision Assist Website

A comprehensive 5-page website for Vision Assist, an AI-powered navigation app designed specifically for people with vision disabilities.

## Features

- **Home Page**: Hero section with key features and testimonials
- **How It Works**: Detailed explanation of AI technology and process
- **Features**: Comprehensive list of app capabilities
- **Accessibility**: Focus on inclusive design and accessibility features
- **Download**: Contact information and app download links

## Technology

- Pure HTML5, CSS3, and JavaScript
- Responsive design for all devices
- WCAG 2.1 AA accessibility compliant
- Modern, accessible UI/UX design
- Mobile-first approach

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rickastley87/vision-assist-nonprofti-web.git
   ```

2. Open `index.html` in your web browser

## Deployment

The website is ready for deployment to any static hosting service:
- GitHub Pages
- Netlify
- Vercel
- AWS S3
- Any web server

## Accessibility

This website follows WCAG 2.1 AA guidelines and includes:
- Screen reader compatibility
- Keyboard navigation
- High contrast mode support
- Semantic HTML structure
- ARIA labels and descriptions
